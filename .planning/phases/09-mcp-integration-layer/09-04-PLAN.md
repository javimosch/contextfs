  <name>Task 2: Create RTKExecutor with fallback logic</name>
  <files>server/mcp/tools/rtk-executor.js</files>
  <action>
    Create `server/mcp/tools/rtk-executor.js` that:
    
    1. Imports dependencies:
       - RTKConfig from '../config/rtk-config.js'
       - ErrorClassifier from './error-classifier.js'
       - NativeExecutor from './native-executor.js'
    
    2. Implements RTKExecutor class:
       - Constructor reads config from RTKConfig
       - Stores nativeExecutor instance for fallback
    
    3. Implements main execute(command, args, options) method:
       - If RTK disabled → return nativeExecutor.execute()
       - If command not in allowlist → return nativeExecutor.execute()
       - Otherwise try RTK first, fallback on Tier 1/2 errors
       - Never fallback on Tier 3 (command errors)
    
    4. Implements executeRTK(command, args, options) method:
       - Map command to RTK args via mapToRTKArgs()
       - Spawn 'rtk' with mapped arguments
       - Collect stdout/stderr
       - On close: classify error, throw if fallback needed
       - Return result with source: 'rtk'
    
    5. Implements mapToRTKArgs(command, args):
       - 'ls' → ['ls', ...args]
       - 'grep' or 'rg' → ['grep', ...args]
       - 'git' → ['git', ...args] (for supported subcommands)
       - Others → [command, ...args]
    
    6. Implements isSupportedCommand(command, args):
       - Check command against allowlist
       - Validate flags are in supported list
       - Return false if any flag unsupported
    
    7. Implements RTKExecutionError class:
       - Extends Error with type, shouldFallback, exitCode properties
       - Used for control flow (distinguish from native errors)
    
    8. Implements tee output saving (ERROR-02):
       - On RTK failure with exitCode !== 0, save full output to file
       - Path: /workspace/.rtk/tee/{timestamp}_{command}.log
       - Include metadata: command, args, exitCode, timestamp
       - Only if config.teeOnError is true
  </action>

    - Saves tee output on failures
