  <action>
    Create contextfs/scripts/rtk-shell-wrapper.sh:

    1. Create the scripts directory if it doesn't exist
    2. Create the wrapper script with:
       - Shebang: #!/bin/bash
       - set -euo pipefail for safety
       - Configuration section reading CONTEXTFS_RTK_ENABLED and CONTEXTFS_RTK_FALLBACK
       - Check if RTK is enabled (default: false)
       - Check if RTK binary exists in PATH
       - Extract command from first argument
       - Map common commands to RTK equivalents (ls→rtk ls, grep/rg→rtk grep)
       - Attempt RTK execution with fallback to native on failure
       - Preserve exit codes

    3. Implementation based on RESEARCH.md pattern:
       ```bash
       #!/bin/bash
       # rtk-shell-wrapper.sh - Wrap commands with RTK fallback logic
       set -euo pipefail

       RTK_ENABLED="${CONTEXTFS_RTK_ENABLED:-false}"
       RTK_FALLBACK_ON_ERROR="${CONTEXTFS_RTK_FALLBACK:-true}"
       COMMAND="${1:-}"
       shift || true

       # If RTK disabled, run native
       if [ "${RTK_ENABLED}" != "true" ]; then
           exec "${COMMAND}" "$@"
       fi

       # Check RTK exists
       if ! command -v rtk &> /dev/null; then
           echo "Warning: RTK not found, using native ${COMMAND}" >&2
           exec "${COMMAND}" "$@"
       fi

       # Map commands to RTK
       case "${COMMAND}" in
           ls) RTK_CMD="rtk ls" ;;
           grep|rg) RTK_CMD="rtk grep" ;;
           *) RTK_CMD="" ;;
       esac

       # Try RTK, fallback on failure
       if [ -n "${RTK_CMD}" ] && [ "${RTK_FALLBACK_ON_ERROR}" = "true" ]; then
           if ${RTK_CMD} "$@"; then
               exit 0
           else
               RTK_EXIT=$?
               echo "Warning: RTK failed (exit ${RTK_EXIT}), using native ${COMMAND}" >&2
               exec "${COMMAND}" "$@"
           fi
       else
           exec "${COMMAND}" "$@"
       fi
       ```

    4. Make script executable: chmod +x contextfs/scripts/rtk-shell-wrapper.sh
  </action>

       ```bash
       #!/bin/bash
       # healthcheck-rtk.sh - Verify RTK installation and basic functionality
       set -euo pipefail

       # Check RTK binary exists
       if ! command -v rtk &> /dev/null; then
           echo "Health check failed: RTK binary not found in PATH"
           exit 1
       fi

       # Check RTK can execute
       if ! rtk --version &> /dev/null; then
           echo "Health check failed: RTK --version command failed"
           exit 1
       fi

       # Get version for success message
       INSTALLED_VERSION=$(rtk --version | grep -oE '[0-9]+\.[0-9]+\.[0-9]+' | head -1)
       echo "Health check passed: RTK ${INSTALLED_VERSION} is functional"
       exit 0
       ```
